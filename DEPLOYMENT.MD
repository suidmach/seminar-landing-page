## How to Set Up the Subdomain in Vercel:

### Step 1: Deploy to Vercel First
1. Push your code to GitHub
2. Connect GitHub to Vercel
3. Deploy
4. You'll get: `your-project.vercel.app`

### Step 2: Add Custom Subdomain
1. In Vercel dashboard ‚Üí Your Project ‚Üí Settings ‚Üí Domains
2. Click "Add Domain"
3. Enter: `seminar.mathewgibeault.ca`
4. Vercel will show you DNS records to add

### Step 3: Update Your DNS
1. Go to wherever you manage mathewgibeault.ca (GoDaddy, Namecheap, etc.)
2. Add a CNAME record:
   - **Name:** `seminar`
   - **Value:** `cname.vercel-dns.com`
   - **TTL:** 3600 (or automatic)
3. Save

### Step 4: Wait & Verify
1. Wait 10-60 minutes for DNS to propagate
2. Visit `https://seminar.mathewgibeault.ca`
3. It should load your landing page!
4. Vercel automatically adds SSL (https)
